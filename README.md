# Project 1 — The Empathy Map
### DecodeLabs UI/UX Design Internship · Batch 2026

---

## Overview

This project is the first milestone of the DecodeLabs Industrial Training Kit for UI/UX Design. The objective is to apply foundational UX research skills, specifically empathy mapping and user persona development, before moving into any visual design work.

**Scenario:** A fitness startup is building a mobile app for busy office professionals who struggle to find time for exercise. The task is to deeply understand this user's pain before designing a single screen.

---

## Deliverables

| Deliverable | Description |
|---|---|
| User Persona | A specific fictional user (age 25–40) with a defined job title, income level, goals, and frustrations |
| Empathy Map | A 4-quadrant map covering what the user Says, Does, Thinks, and Feels |

---

## The 4-Quadrant Empathy Map

| Quadrant | Focus | Example Insights |
|---|---|---|
| **Says** | Outward complaints & excuses | Vocal frustrations to colleagues/family about not having time |
| **Does** | Observable habits | Drinks 4 coffees a day, skips lunch to keep working |
| **Thinks** | Private, unspoken worries | "I'm falling behind my peers." / "I feel unhealthy." |
| **Feels** | Emotional state (end of day) | Guilt, burnout, physical stiffness |

---

## Key Problem Insights

Existing fitness apps fail this audience for three core reasons:

- **Workouts are too long** — no accommodation for a 5-minute activity window
- **Interfaces are too loud/gamified** — clashes with a professional mindset
- **Requires a gym** — inaccessible during a 40+ hour work week

The design must solve for the user who **cannot leave their desk**, not the user who lives at the gym. Solutions explored include micro-workouts and desk-based movement (seated stretches, posture correction, desk-bound exercises).
